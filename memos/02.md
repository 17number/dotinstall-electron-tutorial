# [#02 Electronをインストールしよう](https://dotinstall.com/lessons/basic_electron/36202)

## Node.js のインストール

インストール済みなのでスキップ。環境情報

```bash
$ sw_vers
ProductName:	Mac OS X
ProductVersion:	10.15.2
BuildVersion:	19C57

$ node -v
v12.7.0

$ npm -v
6.10.3
```

## Electron のインストール

```bash
$ npm init -y
$ npm i -D electron
```
